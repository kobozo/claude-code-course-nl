# Claude Code Beginnerstraining

Welkom bij de Claude Code beginnerstraining! In deze cursus leer je stap voor stap werken met Claude Code, de CLI-tool van Anthropic waarmee je AI-ondersteund kunt programmeren.

## Voor wie is deze training?

Deze training is bedoeld voor developers die:

- Basiskennis hebben van de terminal/command line
- Willen leren hoe ze AI kunnen inzetten bij het programmeren
- Nog geen of weinig ervaring hebben met Claude Code

## Wat heb je nodig?

- Een computer met macOS, Linux of Windows (WSL)
- Basiskennis van de terminal
- Een teksteditor (VS Code aanbevolen)
- Een Anthropic account (wordt uitgelegd in module 01)

## Hoe volg je deze training?

1. **Begin bij module 00** en werk de modules op volgorde door
2. **Lees eerst de theorie** (README.md in elke module)
3. **Maak de oefeningen** in de `oefeningen/` map van elke module
4. **Experimenteer!** De beste manier om Claude Code te leren is door het te gebruiken

## Tijdsindicatie

De volledige training kun je in een middag doorwerken. Elke module duurt ongeveer 15-30 minuten, afhankelijk van je tempo en hoeveel je experimenteert.
